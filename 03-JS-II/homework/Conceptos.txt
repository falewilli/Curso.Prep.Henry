For: la expresión for genera un bucle que se repetirá de manera indefinida hasta que se cumpla una determinada condición. En su sintaxis se declara una variable, sobre la cual se repetirá el bucle; luego tenemos la condición, que es lo que el bucle evaluará para saber si debe seguir repitiéndose o no. Dicho de manera más concreta, mientras la condición sea verdadera el bucle se repetirá, y cuando la condición deje de ser verdadera, o lo que es lo mismo, cuando la condición sea falsa, el bucle dejará de repetirse. Y finalmente debemos definir el incremento de la variable, que es lo que hará que la variable se vaya modificando hasta alcanzar la condicón, finalizando así el bucle.

Operador &&: este es el operador "Y" o "AND", se utiliza para evaluar dos condiciones, y de la única manera que devuelve un resultado true es cuando ambas condiciones son ciertas. Con que una de las condiciones no sea cierta, será suficiente para devolver un false. Si la primera condición no es cierta, deja de ejecutarse y devuelve un false ya que ya tiene suficiente información para devolver un resultado.

Operador ||: este es el operador "O" o "OR", se utiliza para evaluar dos condiciones, y devolverá un resultado true simplemente si una de las condiciones es cierta, con eso es suficiente. De la única manera que devuelve un resultado false es cuando ambas las condiciones son falsas. Si la primera condición es cierta, deja de ejecutarse y devuelve un true ya que ya tiene suficiente información para devolver un resultado.

Operador !: este es el operador "NO" o "NOT". Se utiliza por ejemplo para evaluar si dos valores diferentes (!==). El operador NOT devolverá el valor booleano opuesto de lo que se le pasa.
