Array: un array es una colección de datos que se almacenan bajo una sola variable, como por ejemplo un listado. Es una matriz de datos y puede contener tanto números como strings o booleanos, los que se separan por comas y podemos acceder a ellos en cualquier momento y realizar todo tipo de operaciones como quitarlos, reemplazarlos, agregar nuevos, moverlos de posición o de un array a otro, etc. Cada elemento tiene una ubicación dentro del array y es identificado por un ID, comenzando desde el 0, es decir que un array de 10 elementos va a contener IDs del 0 al 9.
